논리회로 수업 진법 프로그램 설계

전제조건.
1.AI가 짠 코드가 아닐 것. (개인적)
2.데이터의 배열사이즈는 없음.
3.입력받은 데이터의 2,8,10,16진수와 그의 r의보수 r-1의 보수를 구하는 것


프로그램 작동 방식
main에서 convert 함수 호출 후 convert에서 입력받은 r값을 
10진수로 변환하여, 각 변환 함수를 호출하여 재변환하여 사용.
main(단순호출)->convert(변환)->BinaryCalc->OctalCalc->DecimalCalc->HexaCalc

main.c 제공기능.
프로그램의 시작점 역할만 수행하며, convert 함수를 호출하여 전체 흐름을 시작한다. 별도의 변환 로직은 포함하지 않고, 실행 진입점으로만 사용한다.

Convert.c 제공기능.
사용자로부터 2, 8, 10, 16진수 중 하나를 입력받아 값을 10진수로 변환한 뒤, 출력할 진수값을 선택한다. 입력된 데이터는각 변환 계산 함수로 전달하며, 양수/음수 선택도 함께 받아 부호를 반영하며, 5 입력 시 프로그램을 종료한다.

BinaryCalc.c 제공기능.
10진수로 변환된 데이터를 바탕으로 2진수, 1의 보수, 2의 보수를 계산하는 기능을 제공한다.
입력값의 절댓값을 기준으로 표현에 필요한 최소 비트 수를 구하고,
부호비트 1개를 포함한 크기의 배열로 결과를 저장한다.
음수 입력의 경우에는 절댓값을 먼저 2진수로 변환한 뒤,
1의 보수와 2의 보수를 차례대로 계산하여 최종 표현을 출력한다.
양수 입력의 경우에도 동일한 방식으로 2진수, 1의 보수, 2의 보수를 함께 확인할 수 있다.

OctalCalc.c 제공기능.
10진수 데이터를 8진수로 변환하고, 각 자리수 기준으로 7의 보수와 8의 보수를 계산하여 출력한다.
음수 입력은 절댓값 기준으로 처리하며, 8진수 변환과 보수 계산 과정을 순서대로 확인할 수 있다.

DecimalCalc.c 제공기능.
10진수 데이터를 그대로 출력하고, 각 자리수 기준으로 9의 보수와 10의 보수를 계산하여 출력한다.
음수 입력은 절댓값 기준으로 처리하며, 보수 계산 과정을 순서대로 확인할 수 있다.

HexaCalc.c 제공기능.
10진수 데이터를 16진수로 변환하고, 각 자리수 기준으로 15의 보수와 16의 보수를 계산하여 출력한다.
음수 입력은 절댓값 기준으로 처리하며, 변환 과정과 결과를 함께 확인할 수 있다.
